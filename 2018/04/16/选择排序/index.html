直接选择排序

原理

初始时在序列中找到最小（大）元素，放到序列的起始位置作为已排序序列；然后，再从剩余未排序元素中继续寻找最小（大）元素，放到已排序序列的末尾。以此类推，直到所有元素均排序完毕。

选择排序代码实现

public void selectSort(int[] arr){
    for(int i = 0;i < arr.length; i++){
        int mix = i;   //mix用来保存最小值索引
        for(int j = i+1;j < arr.length;j++) {
            if(arr[j] < arr[mix]){
                mix = j;//依次比较i后面每一个值与arr[mix]大小，并始终把最小值索引赋值给mix
            }
        }//仅仅在比较完毕后mix与i不等时交换一次，其他时候仅仅记住最小数索引值
        if(mix != i){
            swap(arr,mix,i);
        }
    }
}
private void swap(int[] a,int i,int j){
    int temp = arr[i];
    arr[i] = arr[j];
    arr[j] = temp;
}

堆排序

堆排序原理

基本思想将待排序序列构造成一个大顶堆，此时，整个序列的最大值就是堆顶的根节点。将其与末尾元素进行交换，此时末尾就为最大值。然后将剩余n-1个元素重新构造成一个堆，这样会得到n个元素的次小值。如此反复执行，便能得到一个有序序列了。
a.将无需序列构建成一个堆，根据升序降序需求选择大顶堆或小顶堆;
b.将堆顶元素与末尾元素交换，将最大元素”沉”到数组末端;
c.重新调整结构，使其满足堆定义，然后继续交换堆顶元素与当前末尾元素，反复执行调整+交换步骤，直到整个序列有序。

堆排序代码实现

public void heapSort(int[] arr){
    //建立大根堆，从右向左，从下到上
    for(int i = arr.length/2-1;i>=0;i--){
        adjustHeap(arr,i,arr.length);
    }
    //交换堆顶元素与末尾元素

    for(int j = arr.length-1; j > 0;j--){
        swap(arr,0,j);
        adjustHeap(arr,0,j); //调整堆结构
        //与第一次建立大根堆比较   length减少，反应的是数组长度减少，即堆顶的大元素至数组尾部后不参与堆的调整；而且父子节点的比较是从堆顶开始
    }
}
public void adjustHeap(int[] arr,int i,int length){
    int temp = arr[i];   //提取额当前元素值（根结点）
    for(int k = 2*i+1;k < length;k = 2k+1){//与length比较，判断是否为叶子结点
        if(k+1 < length && arr[k] < arr[k+1]){
            k++;//比较两个子节点的大小，较大的与根结点比较
        }
        if(arr[k] > temp){
            arr[i] = arr[k];//将根节点赋大值，
            i = k;//此时子节点不赋值，查看k=2k+1是否大于length，即此交换的子节点是否为叶子结点，如果还有子节点则循环到下一层比较，没有值的改变，则跳出当前循环；
        }else{
            break;
        }
    }
    arr[i] = temp;//较大子节点位置上浮位置赋值之前提取的根结点值
}
public static void swap(int[] arr,int a ,int b){
    int temp = arr[a];
    arr[a] = arr[b];
    arr[b] = temp;
}

堆排序是整体主要由构建初始堆+交换堆顶元素和末尾元素并重建堆两部分组成。其中构建初始堆经推导复杂度为O(n)，在交换并重建堆的过程中，需交换n-1次，而重建堆的过程中，根据完全二叉树的性质，[log2(n-1),log2(n-2)…1]逐步递减，近似为nlogn。所以堆排序时间复杂度一般认为就是O(nlogn)级。
